# Implementação do Padrão Builder em Python

## Descrição

Este repositório contém a implementação do padrão de projeto criacional **Builder** utilizando a linguagem Python.

O objetivo do projeto é demonstrar, de forma prática, como o padrão Builder pode ser aplicado para organizar a construção de objetos complexos, separando o processo de criação da representação final do objeto.

## Padrão Utilizado

**Builder**

O padrão Builder permite:

- Construir objetos passo a passo  
- Melhorar a legibilidade do código  
- Evitar construtores com muitos parâmetros  
- Facilitar manutenção e futuras extensões  

## Estrutura

- `atividade01.py` — Arquivo contendo a implementação das classes `Computador` e `ComputadorBuilder`.

## Execução

Para executar o projeto:

```bash
python atividade01.py
```
